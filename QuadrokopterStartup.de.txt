Systemstartvorgang aus sicht der Quadrokopter (draft)

1.  Die Quadrokoptermodule werden gestartet.
2.  Die Quadrokoptermodule melden sich bei der API und erhalten ihre id.
3.  Die Quadrokoptermodule scannen den Funkraum und senden das Ergebnis an die API.
4.  Die API weist den Quadrokoptermodulen einen Quadrokopter zu.
5.  Die Quadrokoptermodule warten auf das Startsignal auf dem System Topic und verbinden sich sobald sie start erhalten.
